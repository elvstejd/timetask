## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform

  
## Getting started

```bash
  git clone https://github.com/elvstejd/timetask.git
  cd timetask
```

```bash
  yarn install
```
```bash
  yarn start
```
## Tech Stack

**Client:** React, Styled Components, Axios

**Server:** Node, Express, MongoDB
