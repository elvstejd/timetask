## Features

- Time your tasks
- See how many tasks you've completed in the day
- Get a sound notification when the timer ends

  
## Getting started

```bash
  git clone https://github.com/elvstejd/timetask.git
  cd timetask
```

```bash
  yarn install
```
```bash
  yarn start
```
